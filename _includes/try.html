Or play around with a single line install

Run the following command on a server that has Docker & docker-compose installed on it and you will have a fully running Nebula cluster + a worker with a running example app running on a single server to allow you to easily play around with the API.

curl -L "https://raw.githubusercontent.com/nebula-orchestrator/docs/master/examples/hello-world/start_example_nebula_cluster.sh" -o start_example_nebula_cluster.sh && sudo sh start_example_nebula_cluster.sh
