Play around with a single line install

Run the following command on a Linux server that has Docker & docker-compose installed on it and you will have a complete Nebula cluster (API, RabbitMQ & MongoDB) + a worker with an example app running on it to allow you to easily play around with the API.

curl -L "https://raw.githubusercontent.com/nebula-orchestrator/docs/master/examples/hello-world/start_example_nebula_cluster.sh" -o start_example_nebula_cluster.sh && sudo sh start_example_nebula_cluster.sh
